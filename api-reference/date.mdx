# Date Methods

Lodash 提供了少量与日期和当前时间相关的工具方法，以下为当前文档已覆盖的 API 说明。

---

## now

返回当前时间戳（Unix 时间戳，单位：毫秒）。

该方法等价于 `Date.now()`，在 Lodash 中作为工具函数统一暴露，便于在函数式编程或 FP 模式下使用。

### 函数签名

```ts
function now(): number
```

### 示例

```js
import now from 'lodash/now';

const ts = now();
// => 例如 1700000000000
```
