# Object Methods

Lodash 提供了丰富的对象操作方法，用于对象的创建、合并、遍历、查询与转换等常见场景。本文档将逐一介绍 `api-reference/object` 模块下的核心 API，并给出使用示例与注意事项。

## assign

<Heading level="2">
    assign
</Heading>

将源对象自身的可枚举属性复制到目标对象，后续源对象的属性会覆盖前面源对象的同名属性。

参数：
<BulletedList>
    object (Object): 目标对象
    [sources] (...Object): 源对象
</BulletedList>

返回：
<Paragraph>
    (Object): 返回目标对象
</Paragraph>

示例：
<NumberedList>
    <Mark bold>_.assign({}, { a: 1 }, { b: 2 })</Mark> → { a: 1, b: 2 }
    <Mark bold>_.assign({ a: 1 }, { a: 2, b: 3 })</Mark> → { a: 2, b: 3 }
</NumberedList>

## create

<Heading level="2">
    create
</Heading>

创建一个继承自给定原型对象的对象，可以额外指定自身属性。

参数：
<BulletedList>
    prototype (Object): 新对象的原型
    [properties] (Object): 新对象自身的属性
</BulletedList>

返回：
<Paragraph>
    (Object): 返回新对象
</Paragraph>

示例：
<NumberedList>
    <Mark bold>_.create({ greet() { return 'hi'; } }, { name: 'lodash' })</Mark> 会得到一个可调用 greet 并携带 name 属性的对象
</NumberedList>

## defaults

<Heading level="2">
    defaults
</Heading>

将一个或多个源对象自身的可枚举属性复制到目标对象，仅当目标对象中对应属性为 <Mark color="orange">undefined</Mark> 时才会写入。

参数：
<BulletedList>
    object (Object): 目标对象
    [sources] (...Object): 源对象
</BulletedList>

返回：
<Paragraph>
    (Object): 返回目标对象
</Paragraph>

示例：
<NumberedList>
    <Mark bold>_.defaults({ a: 1 }, { a: 2, b: 3 })</Mark> → { a: 1, b: 3 }
</NumberedList>

## toPairs

<Heading level="2">
    toPairs
</Heading>

将对象自身的可枚举字符串键属性转换为键值对数组。

参数：
<BulletedList>
    object (Object): 要遍历的对象
</BulletedList>

返回：
<Paragraph>
    (Array): 返回键值对数组
</Paragraph>

示例：
<NumberedList>
    <Mark bold>_.toPairs({ a: 1, b: 2 })</Mark> → [['a', 1], ['b', 2]]
</NumberedList>

## toPairsIn

<Heading level="2">
    toPairsIn
</Heading>

将对象自身及继承的可枚举字符串键属性转换为键值对数组。

参数：
<BulletedList>
    object (Object): 要遍历的对象
</BulletedList>

返回：
<Paragraph>
    (Array): 返回键值对数组
</Paragraph>

示例：
<NumberedList>
    <Mark bold>_.toPairsIn(Object.create({ a: 1 }, { b: { value: 2, enumerable: true } }))</Mark> → [['a', 1], ['b', 2]]
</NumberedList>

## findKey

<Heading level="2">
    findKey
</Heading>

遍历对象自身的可枚举字符串键属性，返回第一个使 predicate 返回真值的键。

参数：
<BulletedList>
    object (Object): 要搜索的对象
    [predicate] (Function): 每次迭代调用的断言函数
</BulletedList>

返回：
<Paragraph>
    (string|undefined): 返回匹配的键，未找到则返回 undefined
</Paragraph>

示例：
<NumberedList>
    <Mark bold>_.findKey({ a: 1, b: 2, c: 3 }, v => v > 1)</Mark> → 'b'
</NumberedList>

## forIn

<Heading level="2">
    forIn
</Heading>

遍历对象自身及继承的可枚举字符串键属性，每次迭代调用 iteratee。

参数：
<BulletedList>
    object (Object): 要遍历的对象
    [iteratee] (Function): 每次迭代调用的函数
</BulletedList>

返回：
<Paragraph>
    (Object): 返回原对象
</Paragraph>

示例：
<NumberedList>
    <Mark bold>_.forIn({ a: 1, b: 2 }, (value, key) => console.log(key, value))</Mark> 会依次输出 a 1、b 2
</NumberedList>

## forOwn

<Heading level="2">
    forOwn
</Heading>

遍历对象自身可枚举字符串键属性，每次迭代调用 iteratee。

参数：
<BulletedList>
    object (Object): 要遍历的对象
    [iteratee] (Function): 每次迭代调用的函数
</BulletedList>

返回：
<Paragraph>
    (Object): 返回原对象
</Paragraph>

示例：
<NumberedList>
    <Mark bold>_.forOwn({ a: 1, b: 2 }, (value, key) => console.log(key, value))</Mark> 会依次输出 a 1、b 2
</NumberedList>

## functions

<Heading level="2">
    functions
</Heading>

返回对象自身可枚举的字符串键函数的名称数组。

参数：
<BulletedList>
    object (Object): 要检查的对象
</BulletedList>

返回：
<Paragraph>
    (Array): 返回函数名数组
</Paragraph>

示例：
<NumberedList>
    <Mark bold>_.functions({ a: () => {}, b: 1, c: function() {} })</Mark> → ['a', 'c']
</NumberedList>

## get

<Heading level="2">
    get
</Heading>

根据路径获取对象的值，如果解析值是 <Mark color="orange">undefined</Mark>，返回默认值。

参数：
<BulletedList>
    object (Object): 要查询的对象
    path (Array|string): 属性的路径
    [defaultValue] (*): 默认值
</BulletedList>

返回：
<Paragraph>
    (*): 返回解析到的值
</Paragraph>

示例：
<NumberedList>
    <Mark bold>_.get({ a: { b: 2 } }, 'a.b')</Mark> → 2
    <Mark bold>_.get({ a: { b: 2 } }, 'a.c', 'default')</Mark> → 'default'
</NumberedList>

## has

<Heading level="2">
    has
</Heading>

检查路径是否是对象的直接属性（不检查原型链）。

参数：
<BulletedList>
    object (Object): 要查询的对象
    path (Array|string): 属性的路径
</BulletedList>

返回：
<Paragraph>
    (boolean): 如果路径存在返回 true，否则返回 false
</Paragraph>

示例：
<NumberedList>
    <Mark bold>_.has({ a: { b: 2 } }, 'a.b')</Mark> → true
    <Mark bold>_.has({ a: { b: 2 } }, 'a.c')</Mark> → false
</NumberedList>

## invert

<Heading level="2">
    invert
</Heading>

返回一个新对象，原对象的键和值互换。

参数：
<BulletedList>
    object (Object): 要反转的对象
</BulletedList>

返回：
<Paragraph>
    (Object): 返回反转后的新对象
</Paragraph>

示例：
<NumberedList>
    <Mark bold>_.invert({ a: '1', b: '2' })</Mark> → { '1': 'a', '2': 'b' }
</NumberedList>

## keys

<Heading level="2">
    keys
</Heading>

返回一个由对象自身可枚举字符串键组成的数组。

参数：
<BulletedList>
    object (Object): 要查询的对象
</BulletedList>

返回：
<Paragraph>
    (Array): 返回键名数组
</Paragraph>

示例：
<NumberedList>
    <Mark bold>_.keys({ a: 1, b: 2 })</Mark> → ['a', 'b']
</NumberedList>

## mapKeys

<Heading level="2">
    mapKeys
</Heading>

使用 iteratee 遍历对象的自身可枚举字符串键，返回一个新的对象，其键为 iteratee 的返回值。

参数：
<BulletedList>
    object (Object): 要遍历的对象
    [iteratee] (Function): 每次迭代调用的函数
</BulletedList>

返回：
<Paragraph>
    (Object): 返回新映射对象
</Paragraph>

示例：
<NumberedList>
    <Mark bold>_.mapKeys({ a: 1, b: 2 }, (value, key) => key + value)</Mark> → { a1: 1, b2: 2 }
</NumberedList>

## mapValues

<Heading level="2">
    mapValues
</Heading>

使用 iteratee 遍历对象的自身可枚举字符串键，返回一个新的对象，其值由 iteratee 决定。

参数：
<BulletedList>
    object (Object): 要遍历的对象
    [iteratee] (Function): 每次迭代调用的函数
</BulletedList>

返回：
<Paragraph>
    (Object): 返回新映射对象
</Paragraph>

示例：
<NumberedList>
    <Mark bold>_.mapValues({ a: 1, b: 2 }, value => value * 2)</Mark> → { a: 2, b: 4 }
</NumberedList>

## merge

<Heading level="2">
    merge
</Heading>

递归合并源对象的自身和继承的可枚举字符串键属性到目标对象，同 assign 相比支持深合并。

参数：
<BulletedList>
    object (Object): 目标对象
    [sources] (...Object): 源对象
</BulletedList>

返回：
<Paragraph>
    (Object): 返回目标对象
</Paragraph>

示例：
<NumberedList>
    <Mark bold>_.merge({ a: { b: 1 } }, { a: { c: 2 } })</Mark> → { a: { b: 1, c: 2 } }
</NumberedList>

## omit

<Heading level="2">
    omit
</Heading>

返回一个忽略指定属性的新对象。

参数：
<BulletedList>
    object (Object): 源对象
    [paths] (...(string|string[])): 要忽略的属性路径
</BulletedList>

返回：
<Paragraph>
    (Object): 返回新对象
</Paragraph>

示例：
<NumberedList>
    <Mark bold>_.omit({ a: 1, b: 2, c: 3 }, 'b')</Mark> → { a: 1, c: 3 }
</NumberedList>

## pick

<Heading level="2">
    pick
</Heading>

返回一个只包含指定属性的新对象。

参数：
<BulletedList>
    object (Object): 源对象
    [paths] (...(string|string[])): 要选取的属性路径
</BulletedList>

返回：
<Paragraph>
    (Object): 返回新对象
</Paragraph>

示例：
<NumberedList>
    <Mark bold>_.pick({ a: 1, b: 2, c: 3 }, ['a', 'c'])</Mark> → { a: 1, c: 3 }
</NumberedList>

## result

<Heading level="2">
    result
</Heading>

解析路径上的值，如果是函数则调用它并返回结果，否则直接返回值，未定义时返回默认值。

参数：
<BulletedList>
    object (Object): 要查询的对象
    path (Array|string): 属性的路径
    [defaultValue] (*): 默认值
</BulletedList>

返回：
<Paragraph>
    (*): 返回解析到的值
</Paragraph>

示例：
<NumberedList>
    <Mark bold>_.result({ a: { b: () => 2 } }, 'a.b')</Mark> → 2
</NumberedList>

## set

<Heading level="2">
    set
</Heading>

根据路径设置对象的值，如果路径中间不存在，会自动创建中间对象或数组。

参数：
<BulletedList>
    object (Object): 要修改的对象
    path (Array|string): 属性的路径
    value (*): 要设置的值
</BulletedList>

返回：
<Paragraph>
    (Object): 返回修改后的对象
</Paragraph>

示例：
<NumberedList>
    <Mark bold>_.set({}, 'a.b.c', 1)</Mark> → { a: { b: { c: 1 } } }
</NumberedList>

## transform

<Heading level="2">
    transform
</Heading>

类似 reduce，但结果对象可以自定义初始值，遍历过程中可以直接修改结果对象。

参数：
<BulletedList>
    object (Object): 要遍历的对象
    [iteratee] (Function): 每次迭代调用的函数
    [accumulator] (*): 初始累积值
</BulletedList>

返回：
<Paragraph>
    (*): 返回累积值
</Paragraph>

示例：
<NumberedList>
    <Mark bold>_.transform({ a: 1, b: 2 }, (result, value, key) => { result[key] = value * 2; }, {})</Mark> → { a: 2, b: 4 }
</NumberedList>

## unset

<Heading level="2">
    unset
</Heading>

删除对象中指定路径的属性。

参数：
<BulletedList>
    object (Object): 要修改的对象
    path (Array|string): 属性的路径
</BulletedList>

返回：
<Paragraph>
    (boolean): 如果成功删除则返回 true
</Paragraph>

示例：
<NumberedList>
    <Mark bold>_.unset({ a: { b: 2 } }, 'a.b')</Mark> → true，对象变为 { a: {} }
</NumberedList>

## values

<Heading level="2">
    values
</Heading>

返回一个由对象自身可枚举字符串值组成的数组。

参数：
<BulletedList>
    object (Object): 要查询的对象
</BulletedList>

返回：
<Paragraph>
    (Array): 返回值数组
</Paragraph>

示例：
<NumberedList>
    <Mark bold>_.values({ a: 1, b: 2 })</Mark> → [1, 2]
</NumberedList>
